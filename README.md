\# Art Gallery Website 🎨



This is a simple Art Gallery web application where users can view and purchase handmade drawings.



\## Features



\* User registration page

\* Art gallery product display

\* Add to cart functionality

\* Image gallery of handmade drawings

\* Backend server using Node.js



\## Technologies Used



\* HTML

\* CSS

\* JavaScript

\* Node.js

\* Express.js



\## Project Structure



Art Gallery

│

├── public

│ ├── images

│ ├── index.html

│ ├── register.html

│ └── style.css

│

├── server.js

├── package.json

└── README.md



\## How to Run the Project



1\. Install dependencies



```

npm install

```



2\. Start the server



```

node server.js

```



3\. Open in browser



```

http://localhost:5000

```
